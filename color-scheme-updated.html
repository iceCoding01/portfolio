Portfolio Color Scheme

A professional, clean, and accessible dark theme color palette for your portfolio project

Color Scheme Summary

Dark Theme Conversion Notes

This color scheme represents a modern dark theme with the following key characteristics:

  • Primary Color: Deep teal (#0F766E) - Used for buttons, headers, and branding elements
  • Secondary Color: Mid-tone purple (#7E57C2) - Used for accents and call-to-action elements
  • Background: Blue-black (#020B1A) - Main background color for the site
  • Surface Color: Slightly lighter blue-black (#0B1529) - Used for cards and UI elements on top of the background
  • Text: White (#FFFFFF) - For high contrast against the dark background
  • Dark Accent: Darker teal (#134E4A) - For emphasis and hover states
  • Light Accent: Light teal (#ECFDF5) - For subtle highlights

This dark theme provides a modern, professional appearance while reducing eye strain in low-light environments. The color combination ensures good contrast for accessibility while maintaining visual appeal.
